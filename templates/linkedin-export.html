LinkedIn 최적화 패키지

뱅크에서 파생한 LinkedIn 전 입력 포맷입니다. 반영할 섹션을 켜서 채우고, [복사]로 붙여넣거나 Fill Plan으로 자동 입력하세요.

⚠️ 기본은 복사-붙여넣기(A 모드) — 계정·ToS 리스크 0. LinkedIn은 프로필 편집 공개 API가 없고 무단 자동화는 ToS 위반이라, 자동 입력(B 모드)은 사용자 본인 계정·사용자 직접 로그인·섹션별 승인 안에서만 지원합니다. 자격증명은 저장·전송하지 않습니다.

반영할 섹션 선택 (활성화)

핵심 섹션은 항상 켜져 있고, 나머지는 눌러서 활성화합니다. 활성화한 섹션만 아래에 나타나고 Fill Plan에 포함됩니다.

B 모드 — computer-use 자동 입력 (런타임 지원 시)

사용자 환경이 브라우저 자동화(computer use·chrome use)를 지원하면(예: Claude cowork, Codex 등) 스킬이 대신 입력할 수 있습니다. 없으면 위 복사(A 모드)로 진행하세요.

  • 🔒 로그인은 사용자 본인이 직접 — 스킬은 아이디·비밀번호·2FA를 묻지도·저장하지도·전송하지도 않습니다. 로그인·2FA·보안창을 만나면 정지.
  • 👤 본인 계정만 — 타인 계정·대량 자동화·연결요청/메시지 발송·스크래핑 금지(ToS).
  • ✅ 섹션별 사전 확인 — 각 필드에 무엇을 넣을지 보여주고 승인받은 뒤 입력·저장.
  • 🛡️ 덮어쓰기 보호 — 기존 값을 먼저 읽어 보존/병합 확인 후 반영, 결과 검증.
  • ⚠️ [T3]·미검증 값은 자동 입력에도 넣지 않습니다(날조 금지 승계).
